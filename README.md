## Description
This project focuses on simulating and analyzing fundamental chemical engineering systems. It helps in understanding system behavior under varying operating conditions using computational modeling.

---

## Key Features
- Frictional loss analysis in pipe flow systems using MATLAB scripts.  
- Steady-state and transient analysis of Continuous Stirred Tank Reactor (CSTR) operations.  
- Modeling multicomponent distillation and heat conduction problems.  
- Visualization of system responses for better interpretation.
